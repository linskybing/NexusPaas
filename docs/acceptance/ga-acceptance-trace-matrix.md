# GA Acceptance Trace Matrix

Part of the [GA Acceptance docs](README.md).

This matrix is a compact trace, not a replacement for the source ledgers.
`Done` is row-scoped only. It must not be read as Full GA, V1 external
production launch, or live P0.2-P0.5 completion unless that exact row says so.

## Status Taxonomy

- `Done` - the row's scoped evidence is documented in the linked source files.
- `Open` - required evidence remains missing, partial, local-only, or not yet
  proven in the required live/external scope.
- `Deferred-GPU-Hardware` - the remaining proof specifically requires real GPU
  hardware validation, such as DRA/MPS behavior on GPU nodes, per-device GPU
  telemetry, GPU PID attribution, NVENC/browser WebRTC media, or real GPU
  metrics.

Partial, local, static, fixture, or single-namespace evidence must stay visible
in `Evidence Scope` and must not close live staging, V1 external production
launch, or Full GA rows.

`Deferred-GPU-Hardware` is not equivalent to `Done` and cannot be used for
non-GPU staging, registry, secret, migration, rollback, DR, CI/Sonar, or supply
chain gaps.
